## How to Use

1. **Enter your Player ID** and name in the User Information section
2. **Click "Save User Info"** to store your details
3. **Select a UC package** from available options or enter custom amount
4. **Choose a payment method**
5. **Click "Process UC Top-Up"** to simulate transaction
6. **Review details** in the confirmation popup
7. **Confirm payment** to complete transaction
8. **View your updated balance** and transaction history
